# Proud House — منصة إدارة الأعمال (CRM/ERP)

منصة عربية متكاملة (RTL) لإدارة أعمال شركة توزيع مستحضرات التجميل **Proud House**، في ملف HTML واحد بلا خادم.

## المحتوى
- **الرئيسية** — لوحة مؤشرات حسب الدور الوظيفي.
- **المشتريات** — استيراد ملفات الموردين، أوامر وفواتير الشراء.
- **المخزون** — المنتجات المتوفرة/المعلقة/النافدة.
- **المبيعات والصفقات** — لوحة متابعة (Kanban) + إصدار الفواتير.
- **المالية** — الإيرادات والمصاريف والسيولة.
- **المحاسبة** — قيد مزدوج كامل: دفتر أستاذ، ميزان مراجعة، قائمة دخل، ميزانية عمومية، إقرار ضريبي، أصول ثابتة، موازنات، مراكز تكلفة.
- **العملاء والموردين** — جهات اتصال وشركات.
- **التقارير · الموارد البشرية · إدارة المستخدمين** — مع نظام أدوار وصلاحيات.

## ملاحظة
هذه **نسخة عرض** مملوءة ببيانات وهمية تحاكي شركة عمرها ١٠ سنوات (٢٠١٦–٢٠٢٦). جميع الأسماء والأرقام تجريبية.

## التشغيل
افتح `index.html` في المتصفح مباشرةً — لا يحتاج أي خادم أو تثبيت. للدخول اختر أي دور من قائمة تسجيل الدخول.

## التقنية
HTML/CSS/JavaScript صِرف · SheetJS (Excel) · jsPDF (PDF) · خطوط IBM Plex Sans Arabic.
